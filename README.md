### What it is

Agent Design Atlas reads through and dissects the **50 open-source AI agent frameworks** curated by [awesome-agents](https://github.com/kyrolabs/awesome-agents) (LangChain, AutoGen, CrewAI, smolagents, MetaGPT, Semantic Kernel, and more), distilling them into one **interactive learning atlas**.

Instead of drowning in each project's API docs, it gives you a single lens to see two things clearly: **what parts an agent is made of, and how the same part is built differently across frameworks.**

### What you get

- 🧩 **12 core components** — reasoning loop, planning, memory, tool use, model abstraction, multi-agent orchestration, context engineering… the agent, taken apart into pieces you can study one at a time.
- 🗂️ **50 framework deep-reads** — one consistent write-up per framework: positioning, top-level architecture, a minimal example, and a component-by-component breakdown.
- 🔥 **Interactive comparison matrix** — a "framework × component" heatmap, color-coded by column, filterable by language/paradigm, with hover cross-hairs — see at a glance who implements what.
- 🔬 **Source-level comparison** — on every component page, expand any framework to read its implementation notes and a **real source snippet**.
- 🗺️ **A clickable architecture map** — see how the 12 parts collaborate in layers; click any node to dive in.
- 📚 **Framework field guide** — cards grouped by design paradigm for quick browsing.
- 🔎 Full-text search · 🌗 light/dark themes · 📱 mobile-friendly · ✨ smooth page transitions.

### Three ways to explore

1. **By idea** — build the mental model first: what an agent is, the paradigms, the core design trade-offs.
2. **By component (horizontal)** — pick a part, compare how every framework does it.
3. **By framework (vertical)** — pick a framework, read its full design.

### Who it's for

Developers and researchers who want to **learn agent design from the ground up**, **choose a framework**, or **survey the landscape**.
