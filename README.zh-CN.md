### 这是什么

Agent Design Atlas 把 [awesome-agents](https://github.com/kyrolabs/awesome-agents) 收录的 **50 个开源 AI Agent 框架**（LangChain、AutoGen、CrewAI、smolagents、MetaGPT、Semantic Kernel…）逐一通读、拆解，沉淀成一张**可交互的学习图谱**。

与其淹没在各家文档的 API 细节里，这里用一套统一的视角帮你看清两件事：**一个 Agent 由哪些零件构成，以及同一个零件在不同框架里有哪些不同实现。**

### 你能获得什么

- 🧩 **12 个核心组件** —— 推理循环、规划、记忆、工具调用、模型抽象、多智能体编排、上下文工程…… 把 agent 拆成可独立理解的零件。
- 🗂️ **50 篇框架深读** —— 每个框架一篇统一结构的分析：定位、顶层架构、最小示例、逐组件实现。
- 🔥 **交互式对比矩阵** —— 「框架 × 组件」热力图，按色相分列、可按语言/范式筛选、悬停十字定位，一眼看清谁实现了什么。
- 🔬 **源码级实现对照** —— 在每个组件页展开任意框架，直接看它的实现要点与**真实源码节选**。
- 🗺️ **可点击的架构图** —— 看清 12 个零件如何分层协作，点任意节点直达。
- 📚 **框架图鉴** —— 按设计范式分组的卡片，快速浏览与定位。
- 🔎 全文搜索 · 🌗 明暗主题 · 📱 移动端适配 · ✨ 页面过渡动画。

### 三种逛法

1. **按理念** —— 先建立心智模型：什么是 agent、有哪些范式、核心设计权衡。
2. **按组件（横向）** —— 选一个零件，横向比较所有框架怎么做。
3. **按框架（纵向）** —— 选一个框架，纵向读它的完整设计。

### 适合谁

想**系统学习 Agent 设计原理**、做**框架选型**、或进行**技术调研**的开发者与研究者。
